Nuevos Elementos HTML5

Incorporaciones al estandar

Con el surgimiento de html5 se agregaron nuevas caracteristicas e implementaron nuevas etiquetas, entre los elementos que se incorporaron se encuentran herramientas estadisticas, la posibilidad de reproducir contenido multimedia de forma nativa y ademas se agrego la capacidad de poder realizar distintos tipos de graficos dentro del documento (vectoriales y mapa de bits). Si bien en la actualidad la mayoria de los navegadores soporta estos elementos, podemos corrobar si los mismos son compatible desde el sitio Can I Use

Estadisticas

Cuando necesitamos comparar valores podemos implementar 2 tipos de etiquetas: <progress> que debe utilizar cuando estamos realizando una tarea que aun no fue finalizada, para evaluar el avance durante la misma, ej: la capacitacion; y <meter> que debe utilizar cuando nuestra tarea fue finalizada y contamos con valores definitivos, ej: un examen

API's de Dibujo

Los navegadores actuales permiten la implementacion de etiquetas como <svg> (scalable vector graphics) que permite la realizacion de graficos vectoriales, es decir, imagenes compuestas de lineas y curvas que se calculan matematicamente; la ventaja de las mismas es que no se ven afectadas por los cambios de tamaño o resolucion y utilizan XML como lenguaje.

Por otro lado <canvas> (Lienzo) permite la creacion de graficos raster (mapa de bits) o imagenes pixelares dentro de nuestro documento utilizando como lenguaje JavaScript permitiendonos realizar interacciones con el usuario, gracias a las caracteristicas del mismo.

Preprocesadores

Pseudolenguajes de CSS

Son herramientas que permiten generar hojas de estilos a partir de una sintasis particular, utilizan algun lenguaje de programacion como herramienta de compilacion para realizar dicha tarea. Estos programas poseen las mismas caracteristicas que el lenguaje CSS, pero agregan alguna funcionalidades que el mismo no posee, ademas que su sintaxis suele ser mas corta en comparacion de la hoja de estilos resultante.

LESS (Leaner Style Sheets)

Preprocesador que utiliza Javascript como lenguaje de compilacion, ademas brinda a las hojas de estilos algunas caracteristicas del mismo como la asincronicidad e invocacion de metodos. Para instalarlo como aplicacion, debemos ingresar en la consola el comando de nodejs npm i -g less y para compilar las hojas de estilos deberemos ingresar el comando lessc "input.less" "output.css"

El unico inconveniente que se nos presenta a la hora de trabajar con LESS es que este no posee un metodo de observacion, es decir, que debemos compilar nuestros archivos cada vez que realicemos cambios. Una solucion a esto es utilizar un plugin conocido como Less-Watch-Compiler que lo instalamos utilizando el comando npm i -g less-watch-compiler y para compilar nuestros archivos ingresamos el comando less-watch-compiler "input/" "output/ "file.less"

SASS (Syntactically Awesome Styles Sheets)

Preprocesador que utiliza Ruby como lenguaje de compilacion, a diferencia de LESS, no posee la capacidad de invocar funciones o metodos si estos no fueron previamente declarados. Varia en la forma que se generan los mixins y distingue con sus sintaxis entre los diferentes tipos de elementos.

Para instalarlo debemos ingresar el comando de NodeJS npm i -g sass y para compilar nuestros archivos debemos ingresar el siguiente comando sass --watch "input.scss" "output.css" el cual ofrece un metodo de observacion para una compilacion automatica. Existen 2 formatos de archivo, siendo el primero sass, que no utiliza la mayoria de los signos y se maneja con la tabulacion; y luego esta scss cuya sintaxis es similar a css o less.

Efectos Visuales

Diseñando con CSS

Actualmente CSS permite realizar modificaciones sobre los elementos de la pagina utilizando reglas que toman prestada la sintaxis de aplicaciones de diseño como adobe photoshop, illustrator o animate, entre otras. A diferencia de otros atributos, estos efectos no suelen modificar el flow y por ende no afectan la posicion de los demas elementos de la pagina.

Transformaciones y Transiciones

Una de las caracteristicas incluidas es la capacidad de modificar la apariencia de una etiqueta afectando su contenido; entre los cambios que podemos realizar se encuentran el escalado de elementos, desplazamiento, rotacion, sesgado, e incluso podemos aplicar perspectiva. Si estas modificaciones las realizamos sobre alguna pseudoclase (instancia) incluso podemos agregar estados intermedios a modo de transicion.

Animaciones

Otra de las caracteristicas que podemos encontrar en CSS es la capacidad de crear animaciones utilizando selectores especiales y una regla conocida como @keyframes. Esta herramienta toma de inspiracion el funcionamiento de programas como Adobe Animate tambien conocido como FLASH, donde indicamos estapas clave dentro de nuestra animacion y el programa, en este caso el lenguaje CSS, se encarga de generar los estadios intermedios.

Timing-Function

Al igual que las transiciones, podemos controlar de que manera sucederan las mismas, pudiendo indicar en que momento nuestra animacion sera mas lenta y cuando debe aumentarse la velocidad de reproduccion. Para modificar esta caracteristica debemos utilizar el atributo animation-timing-function y en caso que necesitemos que la misma sea constante podemos utilizar el valor linear.

Direction

En las animaciones podemos controlar la manera en la que sera reproducida nuestra pelicula, por defecto esto se da del from{} al to{}, pero podemos hacer que la misma se ejecute hacia atras utilizando el valor reverse para el atributo animation-direction. Tambien podemos ejecutar ambas utilizando valores como alternate, pero es necesario que nuestra animacion se ejecute mas de una vez.

@keyframes Percentages

Si bien las animaciones se ejecutan automaticamente, son bastante similares a las transformaciones con transiciones, ya que agregacion los estados intermedios entre el principio y el fin; pero en caso de necesitar varios estadios intermedios podemos utilizar selectores de porcentajes donde deben estar si o si el 0%{} y 100%{} para que no existan problemas en la reproduccion.
